# ATM-YALAN
Nuestro programa trata sobre una ATM digital, en donde puedes crearte tu propia cuenta y hasta depositar desde tu casa. 


Que Hace este programa?

Este programa es un ATM digital que permite al usuario:

•	Elegir el idioma (Español o Inglés). 

•	Crear una cuenta nueva proporcionando información básica.

•	Iniciar sesión con un nombre y contraseña.

•	Ver su balance, depositar o retirar dinero.

•	Guardar cuentas nuevas en un archivo para usarlas después.

•	Salir del sistema cuando termine.


Como se usa?

Cuando el programa empieza, primero te pregunta en qué idioma quieres usarlo.

Solo escribes 1 para Español o 2 para Inglés.

Luego aparece un menú con tres opciones:

•	Crear cuenta

•	Iniciar sesión

•	Salir

Si eliges crear cuenta:

•	Escribes tu nombre.

•	Escribes tu edad.

•	Si eres menor de 18, escribes el nombre de tu encargado.

•	Escribes tu número de seguro social (9 dígitos).

•	Eliges el tipo de cuenta (cheque/ahorros).

•	Creas una contraseña.

Tu cuenta queda guardada para poder usarla después.

Si eliges iniciar sesión:

•	Escribes tu nombre.

•	Escribes tu contraseña.

•	Si coinciden, entras a tu cuenta.

Una vez dentro de tu cuenta puedes:

•	Ver tu balance.

•	Depositar dinero.

•	Retirar dinero.

Si no escribes bien tu nombre o tu contraseña, el programa te dice que son incorrectos.

Si eliges salir, el programa se cierra.
